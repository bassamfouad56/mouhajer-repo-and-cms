## ğŸ†˜ Troubleshooting

**Logo not appearing?**
- Check file name matches exactly (case-sensitive)
- Ensure PNG has transparent background
- Verify file is in `public/partners/` folder
- Clear browser cache and refresh

**Logo too large/small?**
- Component will auto-scale to consistent height
- Adjust `MAX_HEIGHT` in LogoMarquee component if needed

**Logo quality issues?**
- Download higher resolution version
- Use SVG format and convert to PNG at 2x size
- Contact brand owner for official press kit
